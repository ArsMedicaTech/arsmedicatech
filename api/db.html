Vector database for RAG (retrieval‑augmented generation) with SurrealDB.

Example usage: `python client = AsyncOpenAI(api_key="your-openai-api-key") vec = Vec(client) msg = asyncio.run(vec.rag_chat("Some query...")) logger.debug(msg) `

async get_context(question, k=4)¶

Retrieve context from the knowledge base for a given question. :type question: str :param question: The question for which context is to be retrieved. :type k: int :param k: The number of nearest neighbors to retrieve (default: 4). :rtype: Optional[List[str]] :return: List of context strings or None if an error occurs.

async init()¶

Initialize the vector database. This method connects to the SurrealDB instance, signs in with credentials, and sets up the necessary schema for the knowledge table. It creates the knowledge table with the required fields and indices. :raises Exception: If the connection or query fails. :rtype: None :return: None

async insert(batch, db)¶

Insert a batch of records into the knowledge table. :type batch: BatchList :param batch: A list of dictionaries, each containing ‘id’ and ‘text’ keys. :type db: Any :param db: An instance of AsyncSurreal connected to the database. :rtype: None :return: None

async rag_chat(question, max_tokens=400)¶

Perform a retrieval-augmented generation (RAG) chat with the OpenAI model. :type question: str :param question: The question to ask the model. :type max_tokens: int :param max_tokens: The maximum number of tokens to generate in the response (default: 400). :rtype: str :return: str: The model’s response to the question.

async seed(data_source, data_type='json')¶

Seed the vector database with knowledge data. :type data_source: str :param data_source: Path to the data source file (JSON or JSONL). :type data_type: str :param data_type: Type of the data source file (‘json’ or ‘jsonl’). :rtype: None :return: None
